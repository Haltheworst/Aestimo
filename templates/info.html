About Aestimo

Built With

Python Flask OpenCV EasyOCR MongoDB JavaScript

Aestimo is an intelligent expense tracking application that uses OCR technology to scan receipts and automatically extract total amounts. It helps users maintain their monthly budgets with real-time spending analysis and visual progress tracking.

Key Features

  • Automatic receipt scanning with OCR
  • Monthly budget tracking and alerts
  • Real-time spending analysis
  • Beautiful dark theme with neon accents
  • Responsive design for all devices
